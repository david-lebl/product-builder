# UI Framework (`mpbuilder.uikit`)

Reusable Laminar components with no domain dependency.

## Default Styles

Default styles are shipped in `uikit.css` (located in `src/main/resources/`).
Import it **before** your application styles:

```html
<link rel="stylesheet" href="uikit.css">
```

The stylesheet provides styles for:

- **Badges**: `.badge`, `.badge-pending`, `.badge-completed`, `.badge-error`, `.badge-warning`, `.badge-info`, `.badge-muted`, `.badge-ready`, `.badge-active`
- **SplitTableView**: `.split-table-view`, `.split-table-header`, `.split-table-search`, `.split-table-filters`, `.filter-chip`, `.split-table-content`, `.split-table-table-wrapper`, `.split-table`, `.split-table-th`, `.split-table-row`, `.split-table-td`, `.split-table-empty`, `.split-table-side-panel`
- **Checkbox Set**: `.checkbox-set`, `.checkbox-set .checkbox-label`
